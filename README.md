                             #  Insurance Data Analysis Dashboard | Power BI

## Project Overview
This project presents an interactive **Insurance Claims Analysis Dashboard** built in Power BI. The dashboard provides a comprehensive overview of insurance policies, premiums, claims, and customer information to help analyze business performance and support data-driven decision-making.

The report allows users to explore insurance data through interactive visualizations, slicers, drill-through pages, and Row-Level Security (RLS).

## Business Objectives
- Analyze premium, coverage, and claim amounts.
- Compare premium amounts across different policy types.
- Monitor claim status distribution.
- Track active and inactive insurance policies.
- Analyze claim amounts across different age groups.
- Enable detailed policy-level analysis using drill-through.
- Restrict report access using Row-Level Security (RLS).

## Dashboard Features
- KPI Cards
  - Total Premium Amount
  - Total Coverage Amount
  - Total Claim Amount

- Interactive Visualizations
  - Premium Amount by Policy Type
  - Claim Status Analysis
  - Claim Amount by Age Group
  - Active vs Inactive Policy Distribution
  - Policy Type Claim Summary Table

- Interactive Slicers
  - Policy Number
  - Claim Number
  - Customer ID

- Drill-Through Page
  - Navigate from summary visuals to detailed policy-level information.

- Row-Level Security (RLS)
  - Created custom security roles.
  - Tested report access using **View As** in Power BI Desktop.

## Calculated Columns
Created calculated columns to improve business analysis:

- **Age Group**
  - Young Adult
  - Adult
  - Elder
- **Policy Status**
  - Active
  - Inactive

## Key Insights
- Travel insurance generated the highest premium amount.
- Active policies represent a larger share than inactive policies.
- Elder customers account for the highest claim amount among age groups.
- Claim status distribution helps identify settled, pending, and rejected claims.
- Drill-through functionality enables detailed investigation of individual policy records.
